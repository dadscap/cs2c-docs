# CS2C-API Docs

Public documentation bundle for CS2C-API.

## Contents

- `docs/` — Markdown pages for the public docs site
- `openapi/openapi.json` — Public market-data OpenAPI spec
- `mkdocs.yml` — Minimal MkDocs configuration

## Start Here

- `docs/index.md`
- `docs/getting-started.md`
- `docs/authentication.md`
- `docs/api-reference.md`

## Notes

- This folder is intended as an export bundle for a separate public docs repo.
- It intentionally excludes internal implementation and operational documentation.
- The bundled OpenAPI spec is trimmed to the market-data surface and excludes account/auth/billing management routes.
